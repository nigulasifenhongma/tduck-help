## 导入用户


### 维护导入参数

如果为全新部署，请先设置用户初始化的密码

```java
sys.user.initPassword  
```

![image.png](https://oss.tduckcloud.com/1651305844738-15783247-38a6-44a9-bc2c-6f8a584f9f13.png)

### 使用excel导入填写用户

点击用户管理、选择需要导入的组织架构，点击导入
